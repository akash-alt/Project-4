Quality Issues:¶

Archive Dataset¶

  1. Some datatypes are wrong (timestamp,retweeted_status_timestamp)
  2. incorrect ratings such as denominator not equal to 10
  3. 745 records have dog name = None
  4. Some records are retweets and replies
  5. Extended URL have some null values
  6. Doggo, Puppo, Pupper and Floofer columns have None values, should be Nan
  7. P1, P2, P3 has no real dog names sometimes
  8. The dataset doesn't have records for all 2354 ids. it only have 2075 ids

Tidiness¶

All retweets and replies should be deleted The image predictions could be condensed to show just the most confident dog breed prediction. All three dataframes can be combined into one single dataframe.

Data Cleaning:¶

Data wrangling, sometimes referred to as data munging, is the process of transforming and mapping data from one "raw" data form into another format with the intent of making it more appropriate and valuable for a variety of downstream purposes such as analytics. A data wrangler is a person who performs these transformation operations. source:https://en.wikipedia.org/wiki/Data_wrangling

breed = []
confidence = []

def get_breed_and_confidence(row):
    if row['p1_dog'] == True:
        breed.append(row['p1'])
        confidence.append(row['p1_conf'])
    elif row['p2_dog'] == True:
        breed.append(row['p2'])
        confidence.append(row['p2_conf'])
    elif row['p3_dog'] == True:
        breed.append(row['p3'])
        confidence.append(row['p3_conf'])
    else:
        breed.append('Not identified')
        confidence.append(np.nan)
